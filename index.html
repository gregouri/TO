⚙ Как подключить свою таблицу — нажмите для инструкции
Шаг 1. В исходной Google-таблице открыть Файл → Поделиться → Опубликовать в интернете. Лист — «Документы сервиса» (не «Лог загрузки»). Формат — CSV. Нажать «Опубликовать».
Шаг 2. Открыть HTML и в начале <script> заменить SHEET_ID и GID на свои.
Альтернатива: в исходной таблице открыть доступ «Все, у кого есть ссылка → Читатель» — тогда сработает обычный URL …/export?format=csv&gid=….
Сбор данных в исходную таблицу уже автоматизирован скриптом importServiceDocsFromDriveFolder (Apps Script): он подтягивает новые XLSX-выгрузки из папки Drive каждый час.
